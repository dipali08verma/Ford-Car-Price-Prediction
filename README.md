# Ford Car Price Prediction

## Project Overview
This machine learning project predicts Ford car prices using Linear Regression.

## Technologies Used
- Python
- Pandas
- NumPy
- Matplotlib
- Scikit-Learn

## Workflow
1. Data Loading
2. Data Cleaning
3. Label Encoding
4. Feature Scaling
5. Train-Test Split
6. Linear Regression Model Training
7. Model Evaluation

## Result
- R² Score: 0.72
- Adjusted R² Score: 0.724
